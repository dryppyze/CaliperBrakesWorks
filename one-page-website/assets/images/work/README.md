# Work Image Slots

Use this folder for real DirectBrakesYYC work photos. Recommended filenames:

- garage-setup.jpg
- front-brakes.jpg
- rear-inspection.jpg
- tools-and-parts.jpg
- finished-job.jpg

After adding images, replace each matching placeholder in `index.html` inside the `job-photo-grid` section with an `<img>` tag, for example:

```html
<img class="site-photo" src="assets/images/work/garage-setup.jpg" alt="Private garage brake service appointment in Calgary">
```

Keep alt text short, specific, and true to the photo.
